# bash_vscode_project_template

## vscode extensions

- Bash IDE
  - Bash language serverからmanを取得したりする
  - `npm i -g bash-language-server`
- Bash Debug
  - debugger
- Bash beautify
  - syntax highlight
- shell format
  - formatter
- shellcheck
  - syntax checker
